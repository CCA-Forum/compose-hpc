Feature Description Status
pce_result This is a built-in variable representing the value returned by the associated function. Supported
pce_all(expression) Universal quantification. Returns true if the expression evaluates to true for each element in the specified array(s). For example, pce_all(u > 0) returns true if the values of all elements in array u are greater than 0. Planned
pce_any(expression) Existential quantification. Returns true if the expression evaluates to true for at least one element in the specified array(s). So pce_any(u > 0) returns true if at least one element in array u is greater than 0. Planned
pce_count(expression) Returns the count of the number of entries in the array that satisfy the expression. For example, pce_count(u > 0) returns the number of elements of array, u, whose values are greater than 0. Planned
